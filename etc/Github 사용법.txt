# 작업 끝, 업데이트
1. bash 터미널에 git add .	// 모든 수정사항을 대기시킨다.
2. bash 터미널에 git commit -m "(수정사항에 대한 내욜)"	// 변경사항에 대한 설명
ex) git commit -m "add poketmon"
3. git push origin main	// 실질적으로 githib에 올리는 코드
